## csye6225-spring2019

Programming Infrastructure Using AWS CloudFormation for Autoscalling 

Steps :

Run `./csye6225-aws-cf-create-auto-scaling-application-stack.sh` STACK_NAME for creating and configuring networking resources using AWS CloudFormation.

Run `./csye6225-aws-cf-terminate-auto-scaling-application-stack.sh` for deleting CloudFormation stack by entering STACK_NAME on prompt.
